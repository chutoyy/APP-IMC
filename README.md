## Installation



Dans un terminal lancer la commande :



`git clone https://github.com/chutoyy/APP-IMC.git`



Se positionner dans le répertoire nouvellement créé :



`cd APP-IMC`



Télécharger les dépendances :



`npm install`



Lancer l'application



`npm start`
